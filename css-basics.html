CSS Introduction

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

What You Should Already Know
Before studying CSS, you should have a basic knowledge of HTML.

What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

CSS Example

Example
body { background-color: lightblue; } h1 { color: white; text-align: center; } p { font-family: verdana; font-size: 20px; }

CSS Solved a Big Problem

HTML was NEVER intended to contain tags for formatting a web page! HTML was created to describe the content of a web page.

When tags like <font>, and color attributes were added to the HTML 3.2 specification, it started a nightmare for web developers. Development of large websites, where fonts and color information were added to every single page, became a long and expensive process.

Types of CSS

There are three main ways to add CSS to HTML documents:

  • Inline CSS: Uses the style attribute inside HTML elements.
  • Internal CSS: Uses a <style> block inside the <head> section of the HTML document.
  • External CSS: Uses an external .css file linked with the <link> tag.

Inline CSS Example

Example
<h1 style="color:blue; text-align:center;">This is a heading</h1>

Internal CSS Example

Example
<head> <style> body {background-color: linen;} h1 {color: maroon;} </style> </head>

External CSS Example

Example
<head> <link rel="stylesheet" href="styles.css"> </head>
Tip: External CSS is the most efficient method for styling large websites.

CSS Syntax

A CSS rule consists of a selector and a declaration block.

CSS Syntax
selector { property: value; property: value; }

CSS Selector

The selector points to the HTML element you want to style.

The declaration block contains one or more declarations separated by semicolons.

Example Explained

Example
p { color: red; text-align: center; }

This paragraph will be red and center-aligned.

  • p is a selector in CSS (it points to the HTML element you want to style: <p>)
  • color is a property, and red is the property value
  • text-align is a property, and center is the property value

CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

CSS Selector Categories

  • Simple selectors (select elements based on name, id, class)
  • Combinator selectors (select elements based on a specific relationship between them)
  • Pseudo-class selectors (select elements based on a certain state)
  • Pseudo-elements selectors (select and style a part of an element)
  • Attribute selectors (select elements based on an attribute or attribute value)

The CSS Element Selector

The element selector selects HTML elements based on the element name.

Example
p { text-align: center; color: red; }

The CSS ID Selector

The id selector uses the id attribute of an HTML element to select a specific element.

Example
#para1 { text-align: center; color: red; }

The CSS Class Selector

The class selector selects HTML elements with a specific class attribute.

Example
.center { text-align: center; color: red; }

CSS Color Names

In CSS, a color can be specified by using a predefined color name:

Example
h1 { color: Tomato; } p { color: DodgerBlue; }

CSS RGB Colors

An RGB color value represents RED, GREEN, and BLUE light sources.

Example
h1 { color: rgb(255, 99, 71); } p { color: rgb(30, 144, 255); }

CSS HEX Colors

A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.

Example
h1 { color: #ff6347; } p { color: #1e90ff; }

CSS Backgrounds

The CSS background properties are used to define the background effects for elements.

Background Color

Example
body { background-color: lightblue; }
This is a light blue background.

Background Image

Example
body { background-image: url('background.png'); }

Background Repeat, Position, and Size

  • background-repeat: repeat, no-repeat, repeat-x, repeat-y
  • background-position: left, right, center, top, bottom, or coordinates
  • background-size: auto, cover, contain, or specific size

CSS Borders

Borders are used to define the outline of an element.

Border Properties

  • border-width
  • border-style (solid, dashed, dotted, double, groove, ridge, inset, outset, none, hidden)
  • border-color
  • border-radius (for rounded corners)
Example
p { border: 2px solid #6366f1; border-radius: 8px; padding: 10px; }
This paragraph has a border and rounded corners.

CSS Margins

Margins are used to create space around elements, outside of any defined borders.

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
  • margin (shorthand)
Example
div { margin: 40px 20px; }
This div has margin.

CSS Padding

Padding is used to create space around an element's content, inside of any defined borders.

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
  • padding (shorthand)
Example
div { padding: 30px 10px; }
This div has padding.

CSS Text

CSS provides several properties for formatting text.

  • color
  • text-align
  • text-decoration
  • text-transform
  • letter-spacing
  • line-height
Example
h1 { color: #6366f1; text-align: center; text-transform: uppercase; }
This is a heading

CSS Fonts

CSS allows you to specify the font of an element using the font-family property.

  • font-family
  • font-size
  • font-style
  • font-weight
  • font-variant
Example
p { font-family: 'Courier New', Courier, monospace; font-size: 18px; font-weight: bold; }
This is a paragraph with custom font.

CSS Text Effects

CSS provides several properties for adding effects to text, such as shadows and spacing.

  • text-shadow
  • letter-spacing
  • word-spacing
Example
h2 { text-shadow: 2px 2px 5px #6366f1; letter-spacing: 2px; }
Text Shadow Example

CSS Box Model

All HTML elements can be considered as boxes. The CSS box model describes the rectangular boxes generated for elements in the document tree and consists of: margins, borders, padding, and the actual content.

  • Content - The actual content of the box, where text and images appear
  • Padding - Clears an area around the content. The padding is transparent
  • Border - A border that goes around the padding and content
  • Margin - Clears an area outside the border. The margin is transparent
Example
div { width: 300px; padding: 25px; border: 5px solid gray; margin: 20px; }
Box Model Example

CSS Display

The display property specifies if/how an element is displayed.

  • block
  • inline
  • inline-block
  • none
  • flex
  • grid
Example
span { display: block; background: #f1f5f9; margin-bottom: 10px; }
This span is displayed as a block.

CSS Position

The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute, sticky).

  • static (default)
  • relative
  • absolute
  • fixed
  • sticky
Example
div { position: relative; left: 30px; top: 10px; }
This div is moved 30px right and 10px down.

CSS Float

The float property is used for positioning and formatting content, for example, let an image float left to the text in a container.

  • float: left;
  • float: right;
  • float: none;
Example
img { float: right; margin-left: 20px; }
This text wraps around the floated image.

CSS Flexbox

The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

Flexbox Elements

To start using the Flexbox model, you need to first define a flex container.

Example
.flex-container { display: flex; }

Flex Container Properties

Property Description
flex-direction Defines in which direction the container wants to stack the flex items
flex-wrap Defines whether the flex items should wrap or not
justify-content Used to align the flex items horizontally
align-items Used to align the flex items vertically

CSS Grid

CSS Grid Layout provides a two-dimensional grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

Example
.grid-container { display: grid; grid-template-columns: auto auto auto; gap: 10px; }

CSS Responsive

Responsive web design makes your web page look good on all devices. Use CSS media queries to change styles based on device characteristics.

Example
@media (max-width: 600px) { body { background-color: lightgreen; } }
Resize your browser window to see the effect!

CSS Animations

CSS allows animation of most HTML elements without using JavaScript or Flash!

  • @keyframes
  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
Example
@keyframes example { from {background-color: red;} to {background-color: yellow;} } div { width: 100px; height: 100px; background-color: red; animation-name: example; animation-duration: 4s; }

CSS Transforms

The transform property applies a 2D or 3D transformation to an element.

  • rotate()
  • scale()
  • translate()
  • skew()
Example
div { transform: rotate(20deg); }
Rotated Div

CSS Variables

CSS variables are custom properties that can be reused throughout a document.

Example
:root { --main-color: #6366f1; } h1 { color: var(--main-color); }
This heading uses a CSS variable for its color.

CSS Units

CSS units define the measurement for properties like width, margin, font-size, etc. There are two main types:

Absolute Units

  • px (pixels) Most common for screens
  • cm (centimeters), mm (millimeters), in (inches), pt (points), pc (picas) Used for print

Relative Units

  • em Relative to the font-size of the element
  • rem Relative to the font-size of the root element
  • % Relative to the parent element
  • vw Relative to 1% of the viewport width
  • vh Relative to 1% of the viewport height
Example
div { width: 50vw; font-size: 2em; margin: 5%; }
Tip: Use relative units for responsive design!

CSS Specificity & !important

Specificity determines which CSS rule is applied when multiple rules match the same element.

Specificity Hierarchy

  • Inline styles (e.g., style="") highest
  • ID selectors (e.g., #id)
  • Class, attribute, and pseudo-class selectors (e.g., .class, [type="text"], :hover)
  • Element and pseudo-element selectors (e.g., div, ::before)

The !important Rule

Adding !important to a CSS property value will override any other declarations, but use it sparingly!

Example
p { color: blue !important; }
Warning: Overusing !important can make your CSS hard to maintain.

CSS Pseudo-classes & Pseudo-elements

Pseudo-classes

Pseudo-classes style elements based on their state.

  • :hover when the mouse is over an element
  • :active when an element is being activated (clicked)
  • :focus when an element has focus
  • :nth-child(n) selects the nth child

Pseudo-elements

Pseudo-elements style parts of an element.

  • ::before insert content before an element
  • ::after insert content after an element
  • ::first-letter style the first letter
  • ::first-line style the first line
Example
a:hover { color: red; } p::first-letter { font-size: 200%; color: #6366f1; }

CSS Media Queries

Media queries are used to apply CSS rules based on device characteristics, such as screen width, height, orientation, and resolution.

Example
@media (max-width: 600px) { body { background-color: lightgreen; } }
Tip: Use media queries for responsive web design!

CSS Filters

The filter property applies graphical effects like blur, brightness, contrast, grayscale, and more to elements (usually images).

  • blur()
  • brightness()
  • contrast()
  • grayscale()
  • sepia()
  • drop-shadow()
Example
img { filter: grayscale(100%); }

CSS Gradients

Gradients are smooth transitions between two or more colors. CSS supports linear and radial gradients.

Linear Gradient

Example
div { background: linear-gradient(to right, #6366f1, #8b5cf6); }

Radial Gradient

Example
div { background: radial-gradient(circle, #6366f1, #8b5cf6); }

Box Shadow

Example

Text Shadow

Example

CSS Best Practices & Organization

  • Use meaningful class and ID names (e.g., .main-header, #footer)
  • Follow a naming convention (e.g., BEM)
  • Organize CSS into sections with comments
  • Keep CSS modular and reusable
  • Minimize the use of !important
  • Use external stylesheets for large projects
Tip: Well-organized CSS is easier to maintain and scale!

CSS Frameworks Overview

CSS frameworks are pre-prepared libraries that make web development faster and easier.

  • Bootstrap Most popular, responsive, lots of components
  • Tailwind CSS Utility-first, highly customizable
  • Bulma Modern, flexbox-based
  • Foundation Responsive, enterprise-level
Note: Frameworks help you build quickly, but learning plain CSS is essential!

CSS Debugging Tools

Modern browsers provide powerful tools for inspecting and debugging CSS.

  • Browser DevTools Inspect elements, view applied styles, and test changes live
  • CSS Validators Check your CSS for errors (e.g., W3C CSS Validator)
  • Extensions Tools like ColorZilla, WhatFont, etc.
Tip: Right-click any element and choose "Inspect" to open DevTools!

CSS Print Styles

Print stylesheets allow you to control how your web page looks when printed.

Example
@media print { body { background: white; color: black; } nav, .sidebar { display: none; } }
Tip: Use @media print to hide navigation and adjust layout for printing.

CSS Accessibility

Accessible CSS ensures your website is usable by everyone, including people with disabilities.

  • Use sufficient color contrast for text and backgrounds
  • Never use color as the only way to convey information
  • Use :focus styles for keyboard navigation
  • Ensure readable font sizes and line heights
  • Hide content visually but keep it accessible with .sr-only classes
Example: Focus Style
a:focus { outline: 2px solid #6366f1; outline-offset: 2px; }
Tip: Test your site with a keyboard and screen reader!
